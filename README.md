# B.Tech IISc Website

This is a Next.js project implementing the B.Tech IISc website design.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Structure

- `src/components/Header.tsx`: The main header component.
- `src/components/Header.module.css`: Styles for the header.
- `src/app/page.tsx`: Homepage with Hero section.
- `src/app/layout.tsx`: Root layout tailored for the design.
- `src/app/globals.css`: Global styles and variables.
